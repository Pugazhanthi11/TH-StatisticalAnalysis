# Statistical Analysis

For this, I performed statistical analysis on the COVID-19 dataset created during the previous tasks.

I started by generating summary statistics to understand the overall distribution of the data. After that, I used correlation analysis to examine the relationship between important variables such as cases, deaths, recoveries, and active cases.

## Tools Used

- Python
- Pandas
- SciPy

## Files

- statistical_analysis.py
- featured_covid_data.csv
- correlation_results.csv

## What I Learned

This helped me understand how statistical methods can be used to explore relationships within a dataset. Correlation analysis provided useful insights into how different COVID-19 metrics are connected.
